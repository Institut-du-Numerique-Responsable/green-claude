🌱 Green Claude

Sobriété numérique pour Claude Code : des requêtes IA frugales, et du code éco-conçu selon le RGESN 2024 et le GR491.

Voir sur GitHub Installer

Une double vocation

L'impact environnemental de « coder avec l'IA » a deux faces — Green Claude agit sur les deux :

1. Utiliser Claude sobrement

Modèle proportionné à la tâche, données envoyées limitées, cache local des réponses, heures creuses, et 12 pratiques de sobriété inspirées de Boris Cherny (créateur de Claude Code). Moins de tokens = moins d'énergie.

2. Produire du code sobre

35 règles d'éco-conception alignées sur les 9 familles du RGESN 2024, avec renvoi aux critères officiels. Elles auditent le code (--eco-check) et guident Claude pour qu'il le conçoive sobre dès le départ.

En 30 secondes

git clone https://github.com/Institut-du-Numerique-Responsable/green-claude.git
cd green-claude && ./install.sh

# Question simple → petit modèle
green-claude --complexity simple "Explique cette regex"

# Audit d'éco-conception, sans appel IA (zéro token)
green-claude --file src/app.js --eco-check

Ce que contient le projet

  • Un wrapper green-claude pour le CLI Claude Code : choix du modèle, limites de taille, cache, heures creuses ;
  • 35 règles d'éco-conception couvrant les 9 familles du RGESN 2024 — dont l'Algorithmie et l'IA frugale — reliées au GR491 ;
  • 12 pratiques de sobriété d'usage de Claude Code, sourcées et expliquées (pourquoi c'est sobre, quel geste faire) ;
  • Un audit de code qui détecte les motifs contraires à l'éco-conception (SELECT *, bibliothèques lourdes, autoplay…).

Pourquoi c'est important

Chaque requête à un grand modèle consomme de l'énergie — et le code généré sera exécuté des millions de fois pendant des années. La sobriété numérique se joue donc aux deux bouts : pendant le développement assisté par IA, et dans le logiciel produit. Green Claude outille les deux, en s'appuyant sur les référentiels publics français et internationaux.

Un projet open source (MIT) de l'Institut du Numérique Responsable.
Cette page applique les règles qu'elle promeut : zéro JavaScript, zéro requête externe, polices système, mode sombre natif — moins de 6 Ko.
