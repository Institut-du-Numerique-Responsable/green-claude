🌱 Green Claude

Un skill pour Claude Code qui guide Claude vers un code éco-conçu selon le RGESN 2024 et le GR491 — installé une fois, appliqué automatiquement.

Voir sur GitHub Installer

Ce que le skill fait pour toi

Pendant que Claude code

35 règles d'éco-conception alignées sur les 9 familles du RGESN 2024, avec renvoi aux critères officiels : Claude les applique de lui-même en écrivant du code, sans qu'on le lui demande à chaque fois.

Sur demande, un audit

« Audit éco-conception de ce fichier » lance un script déterministe (grep sur les règles), zéro coût de raisonnement, qui détecte les motifs contraires à l'éco-conception (SELECT *, bibliothèques lourdes, autoplay…).

En 30 secondes

git clone https://github.com/Institut-du-Numerique-Responsable/green-claude.git
cd green-claude && ./install.sh

Le skill s'installe dans ~/.claude/skills/green-claude et se charge automatiquement dans les sessions Claude Code suivantes — rien à taper ensuite. Tape /green-claude pour voir la checklist complète.

Ce que contient le projet

  • Un skill Claude Code, auto-invoqué dès qu'il s'agit d'écrire ou revoir du code ;
  • 35 règles d'éco-conception couvrant les 9 familles du RGESN 2024 — dont l'Algorithmie et l'IA frugale — reliées au GR491 ;
  • 14 pratiques de sobriété d'usage de Claude Code inspirées de Boris Cherny (créateur de Claude Code), sourcées et expliquées ;
  • Un hook optionnel (cache local des réponses, avertissement heures creuses) pour ce qu'un skill ne peut pas faire lui-même — proposé à l'installation, jamais imposé.

Pourquoi c'est important

Le code généré par l'IA sera exécuté des millions de fois, pendant des années, chez des milliers d'utilisateurs : c'est là que se joue l'essentiel de l'empreinte du « coder avec l'IA ». Green Claude outille ce moment précis, en s'appuyant sur les référentiels publics français et internationaux plutôt que sur des règles maison.

Un projet open source (MIT) de l'Institut du Numérique Responsable.
